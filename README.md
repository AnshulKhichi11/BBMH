# 🏥 **BBMH Hospital Website**  

A modern **hospital management website** designed to streamline **patient appointments, doctor schedules, and hospital administration** with a user-friendly interface.  

---

## 🚀 **Features**  

✅ **Patient Registration & Login** (JWT-based authentication)  
✅ **Doctor Profiles & Appointments**  
✅ **Online Appointment Booking System**  
✅ **Hospital Services & Departments Overview**  
✅ **Emergency Contact & Helpline**  
✅ **Admin Dashboard for Managing Doctors & Patients**  
✅ **Mobile-Friendly & Responsive UI**  

---

## 🛠 **Tech Stack**  

### **Frontend:**  
- React.js (Next.js for SEO & SSR)  
- Tailwind CSS / Bootstrap  
- Redux Toolkit (State Management)  

### **Backend:**  
- Node.js & Express.js  
- MongoDB with Mongoose / PostgreSQL  
- JWT for authentication  
- Cloudinary for image storage  

### **Deployment:**  
- **Frontend:** Vercel / Netlify  
- **Backend:** Render / Heroku / DigitalOcean

## ⚡ **Installation & Setup**  

### 1️⃣ Clone the repository  
```bash
git clone https://github.com/your-username/bbmh-hospital-website.git
cd bbmh-hospital-website
```

### 2️⃣ Install dependencies  
```bash
# Backend
cd server
npm install

# Frontend
cd client
npm install
```

### 3️⃣ Set up environment variables  
Create a `.env` file in the **server folder**:  
```ini
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
CLOUDINARY_URL=your_cloudinary_api_url
```

### 4️⃣ Start the development server  
```bash
# Backend
cd server
npm run dev

# Frontend
cd client
npm start
```

### 5️⃣ Open in browser  
```
http://localhost:3000
```
